---
description: テスト実行→失敗時自動修正。「テスト直して」で使う
allowed-tools: Bash(npx vitest*), Read, Edit, Glob, Grep
---

テストを実行して、失敗があれば自動修正する。

1. `npx vitest run $ARGUMENTS` でテスト実行
2. 失敗したテストのエラーメッセージを分析
3. テストファイルと対象の実装ファイルを読む
4. 実装を修正（テストの期待値が正しい前提）
5. 再度テスト実行して全パス確認
6. 修正内容を簡潔に報告

修正は最小限に。テスト自体がおかしい場合はユーザーに確認。
